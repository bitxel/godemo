# Code of Conduct

## Our Pledge

We as maintainers and contributors pledge to make participation in this project
a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and constructive in discussions
- Gracefully accepting feedback
- Focusing on what is best for the community

Examples of unacceptable behavior include:

- Harassment or insulting language
- Personal attacks or discrimination
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of
conduct and may remove or edit comments, commits, code, and other contributions
that are not aligned with this code of conduct.

## Scope

This code of conduct applies in all project spaces and when someone
is officially representing the project in public spaces.

## Contact

For conduct concerns, please contact project maintainers through the repository
security contact channel described in `SECURITY.md`.
